# 🏠 Kakinada House Finder (AI Powered Web App)

A modern AI-powered house listing and discovery platform built using **Google AI Studio**, **Supabase**, and deployed on **Vercel**.  
This project allows users to browse houses for rent in Kakinada with details like location, rent, images, and contact options.

## 🗄️ Database Structure (Supabase)

Table: `house`

| Column   | Type   | Description              |
|----------|--------|--------------------------|
| title    | text   | Name of the house        |
| location | text   | Area in Kakinada         |
| rent     | int8   | Monthly rent in INR      |
| image    | text   | House image URL          |

---

## ⚙️ How It Works

1. User opens the web app
2. App fetches data from Supabase API
3. Houses are displayed in card layout
4. Users can search/filter by location or rent
5. Data updates automatically from backend
